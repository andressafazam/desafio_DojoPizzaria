##  Dojo Pizzaria 
### Sistema para gerenciamento de uma pizzaria feito em JavaScript

#### Esse sistema possui as seguintes funções:
####  - Homepage com instruções
![GitHub Home](/images/1-Home.png)

####  - Função que mostra todas as pizzas do catálogo
![GitHub listarPizzas](/images/2-catalogoPizzas.png)

####  - Função que busca uma pizza pelo nome
![GitHub buscarPizzaPorNome](/images/3-buscarPorNome.png)

####  - Função que busca pizzas pela categoria
![GitHub listarPizzasPorCategoria](/images/4-listarPorCategoria.png)

####  - Função que adiciona uma nova pizza no catálogo
![GitHub adicionaNovaPizza](/images/5-adicionandoPizza.png)

####  - Função que mostra todos os pedidos
![GitHub listarPedidos](/images/6-listaPedidos.png)

####  - Função que adiciona um novo pedido
![GitHub adicionaNovoPedido](/images/7-adicionandoPedido.png)


###### Nota: Rodar o comando NPM INSTALL para instalar todas as dependências do projeto
